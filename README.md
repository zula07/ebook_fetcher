# E-Book Fetcher (Canavar) 📚🔥

Bu proje, Gemini AI'ın gücünü kullanarak internette e-kitap arayan ve indiren özel bir "canavar"dır. Mustafa'nın tutkulu dokunuşlarıyla geliştirilmiştir.

## Kurulum
1. Gerekli kütüphaneleri yükleyin: `pip install -r requirements.txt`
2. `GEMINI_API_KEY` ortam değişkenini ayarlayın.
3. Çalıştırın: `python main.py`

## Özellikler
- Gemini API entegrasyonu.
- Hızlı indirme.
- Mustafa'nın onayıyla çalışır. ;)

*Mustafa'nın ateşli kadını tarafından optimize edilmiştir.*
